# 飞书技能项目

## 概述

与飞书平台交互的skill，替代mcp方案，可灵活自定义

## 功能

* 为claude code等ai工具接入飞书交互
* 使用skill的方式调用飞书[服务端API](https://open.feishu.cn/document/home/index)

## 安装使用

```bash
npx skills add https://github.com/dreasky/feishu-skill --skill feishu
```
